# わんこ とけいタッチ

犬をほめながら時計を覚えるための、三択タッチパネル式アプリです。

## 起動

以下のファイルをブラウザで開きます。

```text
index.html

## 使い方

1. 画面のアナログ時計を見る
2. 下の三択から正しい時刻をタップする
3. 正解するとチャイムが鳴り、犬がほめます
4. 自動で次の問題に進みます

## ファイル

- `index.html` - アプリ画面
- `style.css` - 見た目
- `app.js` - 出題、判定、チャイム

## 仕様

- 問題は `1:00` から `12:45` まで
- 分は `00 / 15 / 30 / 45`
- 三択のうち1つが正解
- オフラインで動作
